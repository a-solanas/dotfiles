# WezTerm Keybindings

## Font Size
- `CMD + =` / `CMD + +` - Increase font size
- `CMD + -` - Decrease font size
- `CMD + 0` - Reset font size

## Window & Pane Management
- `CMD + D` - Smart split (splits along longest dimension)
- `CMD + SHIFT + D` - Split vertically (top/bottom)
- `CMD + OPT + D` - Split horizontally (left/right)
- `CMD + S` - Close current pane (keeps last pane open)
- `CMD + W` - Close pane/tab
- `CMD + T` - New tab
- `CMD + 1-9` - Switch to tab 1-9
- `OPT + ENTER` - Toggle fullscreen

## Navigation
- `CMD + SHIFT + [` - Previous tab
- `CMD + SHIFT + ]` - Next tab

## Other
- `CMD + SHIFT + R` - Reload configuration
- `CMD + SHIFT + L` - Show debug overlay
- `CMD + SHIFT + X` - Copy mode (visual selection)
- `CTRL + SHIFT + C` - Open this cheatsheet
- `CTRL + SHIFT + P` - Command palette

## Mouse
- **Right click** - Paste clipboard
- **SHIFT + Right click** - Vertical split
- **CMD + Right click** - Horizontal split
